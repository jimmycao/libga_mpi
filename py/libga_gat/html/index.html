Parameters explained

Problem definition

Number of objectivesOne ore more-dimensional problems are supported. This mainly affects the algorithm to be chosen and the type of visualization.
Number of variablesMust have at least one variable.

Constraints

Min / MaxChoose this if each variable in your problem definition has the same minimum and maximum real-valued constraint.
From fileLoad a CSV file with constraint pairs for each variable.

Definition

Python moduleSelect a .py file which contains your problem definition. Examples are provided in the test_problems.py file.
Function nameThe objective function's name (e.g. rastrigin_6, kursawe).

View results

In two- and three-dimensional plots You can configure which objective is displayed on which axis. It is also possible to display e.g. only two objectives of a higher-dimensional objective function. Note, that picking solution candidates and displaying their properties is only supported in the 2D plot.

Population Number of solution candidates including archive or elite.
